== Description ==

The protocols.io Wordpress Publications Widget displays completely customizable and responsive publications from protocols.io, any user or any group. It uses the protocols.io API and stores results in the cache.

**Features**

 - Completely Customizable
 - Responsive and mobile optimized
 - Display publications from protocols.io, any user or any group
 - Sort publications by views, dates, etc.

The protocols.io Publications Widget uses protocolsio API. You have to create your own public access token on protocols.io. All the data received from protocols.io via API is cached in WordPress database for a short period to provide front end optimization. You can update cached data.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/protocols-io-publications-widget` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Appearance â€“ Widgets to configure the protocols.io Publications Widget
